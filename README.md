# SmartShield: Phishing Detection & Awareness System

## Overview
SmartShield is a lightweight phishing detection and awareness project designed to help users identify malicious emails using a risk-based scoring model.

## Features
- Email classification (Safe / Suspicious / Phishing)
- Risk scoring system
- Interactive browser-based checker
- Real-world phishing examples
- Business-friendly explanations
